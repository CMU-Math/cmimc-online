Programming Contest

CMIMC Programming is a brand new high school programming contest! It's run by CMU students, the same team behind the CMIMC math competition. This year's contest will be held on January 30th, 2021. Due to COVID-19, it will be entirely online.

The contest consists of an AI Round and an Optimization Round. See the info page for more details.

This is a team contest, with up to 3 people per team. Make sure you register a team by January 16th, 2021! See the registration page for instructions to register.
